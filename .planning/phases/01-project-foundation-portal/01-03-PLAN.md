Create `src/middleware.ts`. Next.js requires this exact filename and `export function middleware`.

```typescript
import { NextResponse } from 'next/server'
import type { NextRequest } from 'next/server'

export function middleware(request: NextRequest) {
  const payloadToken = request.cookies.get('payload-token')

  if (!payloadToken) {
    return NextResponse.redirect(new URL('/', request.url))
  }

  return NextResponse.next()
}

export const config = {
  matcher: ['/flaechen/:path*'],
}
```

Key rules:
- Matcher covers ONLY `/flaechen` — homepage handles its own auth state via SSR
- Do NOT include `/register` — must be reachable without a token
- Do NOT include `/api` — breaks Payload's internal endpoints
- Do NOT include `/admin` — handled by Payload's own access config (Task 3)

After creating `src/middleware.ts`, delete `src/proxy.ts` — it is superseded and was never active.
